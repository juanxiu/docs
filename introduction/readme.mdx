## 워크숍 소개

이 워크숍에서는 Amazon Bedrock에서 에이전트를 사용할 애플리케이션의 설정 프로세스를 자세히 설명합니다.
여기에는 S3 버킷, 지식 베이스, 액션 그룹, Lambda 함수 및 Cloud9 환경 설정이 포함됩니다. 사용자 인터페이스에는 Streamlit 프레임워크를 사용할 것입니다.
에이전트는 특정 매개변수에 따라 최고 수익 기업 포트폴리오를 동적으로 생성하고 연방 공개시장위원회 보고서에 대한 Q&A 기능을 제공하도록 설계되었습니다.
이 워크숍에서는 이메일 전송 방법도 포함되어 있지만 활성화되지는 않습니다.

Level : 300
예상 소요 시간 : 2시간
사용리전 : us-west-2

##  워크숍 구조 
이 워크숍은 총 6개의 모듈로 구성됩니다.
- Amaazon S3 버킷 설정
- AWS Lambda 함수 설정
- Amazon Bedrock 에이전트 및 작업 그룹 설정
- Amazon Bedrock 에이전트로 지식 베이스 설정
- 구성 테스트
- Cloud9 IDE 설정
