## 단계 8 : IDE 생성 및 배포
1. Cloud9 관리 콘솔을 탐색하세요. 그런 다음 Create Environment 을 선택합니다.

2. 여기에서는 각 필드에 다음 값을 입력합니다.
- Name: Bedrock-Environment (원하는 이름 입력)
- Instance type: t3.small
- Platform: Ubuntu Server 22.04 LTS
- Timeout: 1 hour

3. 완료되면 화면 하단의 Create 버튼을 선택하세요. 환경이 가동되는 데 몇 분 정도 걸립니다. 리소스 부족으로 인해 Cloud9을 가동하는 중에 오류가 발생하는 경우 인스턴스 유형으로 t2.micro를 선택하고 다시 시도할 수도 있습니다.
>이 문서 작성 시점에 Cloud9 환경에는 Python 3.10.12 버전이 있습니다.

4. Cloud9 환경으로 다시 이동한 다음 방금 생성한 Cloud9 옆에 있는 Open 링크를 클릭합니다. 이제 Streamlit 앱을 설정할 준비가 되었습니다!
