## 단계 6 : Alias 만들기
1. Alias(별칭)를 만들고 원하는 이름(예. PortfolioCreator-1)을 입력하세요. Alias가 생성되고 새로운 버전이 PREPARED 상태로 생성된 것을 볼 수 있습니다. 에이전트 Alias ID 를 반드시 복사하여 저장하세요. 9단계에서 이 정보가 필요합니다.
<img width="1481" alt="Image" src="https://static.us-east-1.prod.workshops.aws/public/84fc3c8f-f604-4a52-b108-1e4ddb2e9fd1/static/create_alias.png" />


2. 그런 다음 에이전트에 대한 Agent Overview 설정까지 위로 스크롤하고 에이전트 ARN을 복사합니다(9단계에서 필요하므로 Agent ID 도 별도로 복사합니다). 이전에 3단계에서 생성한 Lambda 함수 PortfolioCreator-actions의 리소스 정책에 에이전트 ARN을 추가합니다.
